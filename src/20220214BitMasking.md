# 비트마스킹 BitMasking

- 컴퓨터는 내부적으로 모든 자료를 이진수로 표현
이와 같은 특성을 이용해 정수의 이진수 표현을 자료구조로 쓰는 기법

- 집합의 표현
    - 원소의 개수가 N인 집합이 있다고 하면, 각각의 원소를 0번 부터 N-1번 까지 번호를 부여하고, 번호에 해당하는 비트가 1이면 원소가 포함, 0이면 원소가 불포함

    - a|b a와 b의 합집합
    - a&b a와 b의 교집합
    - a&~b a에서 b를 뺀 차집합
    - a^b a와 b 중 하나에만 포함된 원소들의 집합
